Buttons

<button> has a default definition. You can use the class .button on any <a> tag to make a link look like a call to action. By using modifiers you can generate flat and call to action buttons.

Element Code snippet Modifier class
Link button <a class="button">Action</a> default state
Link Call <a class="button button--call">Action</a> .button--call

Size modifiers

Add .button--small or .button--large to modify button appearance. This just changes font-size and paddings, so you can use it on any button/action.

Element Code snippet Modifier class
<button class="button--large">Action</button> .button--large
<button class="button--call">Action</button> no size modifier
<button class="button--call button--small">Action</button> .button--small

State modifiers

There are also state modifiers defined. To disable a button, you can either use the disabled attirbute or the button--disabled class. Which is specially useful for links that look like buttons.

Notice that disabled buttons have cursor:default and pointer-events:none defined in the css file.

Element Code snippet Modifier class
<button class="button button--flat" disabled>No action</button> no modifier (disabled via attribute)
<button class="button button--flat button--disabled">No action</button> .button--disabled

Inputs

Each <input> has a number of built-in states and modifiers. This applies to [type="text"], [type="password"] and [type="email"]

Element Code snippet Modifier class
<input type="text" placeholder="Text input"> default state
<input type="text" placeholder="Text input" disabled> disabled state
<input type="text" class="input--error" placeholder="Error input"> .input--error

Also, a general style modifier is defined for inputs to behave like a single line. Just add .input--line and there you go.

Modifiers

Element Code snippet Modifier class
<input class="input--line" type="text" placeholder="Text input"> input--line
<input class="input--line" type="text" placeholder="Text input" disabled> input--line disabled state
<input type="text" class="input--line input--error" placeholder="Error input"> input--line input--error
